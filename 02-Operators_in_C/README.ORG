* Arithmetic Operators in C
** ARITHMETIC OPERATORS
- ~+, -, *, /, %~
|---+----------------|
| + | addition       |
| - | Subtraction    |
| * | Multiplication |
| / | Division       |
| % | Modulus        |

All are binary operators, means two operands are required to perform operation.
- For example:
  - A + B
  - Op1 + Op2
#+begin_src c
#include <stdio.h>

int main(){
  int a, b;
  printf("Enter the first number: ");
  scanf("%d", &a);
  printf("Enter the second number: ");
  scanf("%d", &b);

  printf("a/b=%d\n", a/b);
  printf("a%%b=%d\n", a%b);
  return 0;
}
#+end_src

** OPERATOR PRECEDENCE AND ASSOCIATIVITY
|-----------+---------------+------------|
| Operators | Associativity | Precedence |
| *,/,%     | Left to right | Highest    |
| +,-       | Left to right | Lowest     |
NOTE: Associativity is used only when two or more operators are of same precedence.
- For example: +,-
Same precedence therefroe we use associativity.
#+begin_src c
#include <stdio.h>

int main(){
  int a=2, b=3, c=4, d=5;
  printf("a*b/c=%d\n", a*b/c);
  printf("a+b-c=%d\n", a+b-c);
  printf("a+b*d-c%%a=%d", a+b*d-c%a);
  return 0;
}
#+end_src
